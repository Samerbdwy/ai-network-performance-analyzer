# 📊 AI Network Performance Analyzer

An intelligent network quality testing tool that measures latency, jitter, and packet loss to any target using ICMP ping sweeps, with AI-powered performance insights.

---

## 🎯 What It Does

Enter any IP address or hostname → AI instantly tells you:

- **LATENCY:** Response time in milliseconds
- **JITTER:** Variation in latency (stability)
- **PACKET LOSS:** Percentage of lost packets
- **SUCCESS RATE:** Overall connection reliability
- **AI INSIGHTS:** Health assessment and recommendations

Works with routers, websites, DNS servers, or any network target.

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Backend** | Python + Flask |
| **AI Engine** | Google Gemini 2.5 Flash |
| **Network Testing** | ICMP Ping (ping3) |
| **Frontend** | HTML5, CSS3, JavaScript |
| **Data Storage** | SQLite (historical metrics) |

| Feature | Description |
